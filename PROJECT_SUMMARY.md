## 📊 技术栈总结

### 后端 (Go)
- **框架**: Gin 1.9
- **ORM**: GORM 1.25
- **数据库**: MySQL 8.0
- **认证**: JWT + bcrypt
- **文档**: Swagger
- **文件存储**: 本地/阿里云 OSS/腾讯云 COS
- **数据处理**: Excel (excelize)

### 前端 (React)
- **框架**: React 18
- **语言**: TypeScript
- **UI 库**: Ant Design 5
- **状态管理**: Zustand
- **HTTP 客户端**: Axios
- **构建工具**: Vite
- **样式**: TailwindCSS
